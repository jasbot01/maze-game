
# General Concept

AI Rescue Bot / Robot Navigator

- Concept:
  - The player controls a small robot navigating a *some place* and must find/reach a charging station before it runs out of power.
  - Robot has limited battery.
  - Robot must navigate a grid maze.
  - Robot reaches charging station before power hits 0.
  - Grid-based game.
